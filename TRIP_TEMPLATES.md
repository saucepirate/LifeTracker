# LifeTracker — Trip Templates Reference

All templates available in the Trips module, organized by type.

## Trip Planning Project Templates

Used when creating a Planning project from a trip workspace. Select via the **Planning** tab → **Create planning project**. Filters help narrow to the best match; a *Recommended* badge auto-highlights based on days until departure.

## Packing List Presets

Applied from the **Packing** tab → **Templates ▾** → **Browse with filters** or the preset list. Each preset is structured into four sections that map directly to the multi-list packing model:

| Section | What goes here |
|---------|---------------|
| 🤝 **Shared** | One item per group — gear, documents, chargers, sunscreen, group snacks |
| 👤 **All Travelers** | Each person brings their own — clothing, toiletries, personal electronics |
| ♂ **Men** | Gender-specific: grooming, attire variations, formalwear |
| ♀ **Women** | Gender-specific: grooming, attire variations, swimwear, toiletries |

When "Per-traveler lists" is selected on apply, Shared items go to the **Shared** list; All Travelers / Men / Women items are routed to each personal list based on traveler composition.

## Template Filter Reference

When creating a planning project from a trip, four filter dimensions narrow the template grid. **Trip type** is the primary signal for surfacing specialized templates; **Trip length** is a secondary narrowing filter.

| Filter | Values |
|--------|--------|
| **Planning window** | Any · 1 Month · 3 Months · 6 Months · 1 Year |
| **Destination** | Any · Domestic · International |
| **Trip type** | Any · General / Mixed · Sightseeing / City · Beach / Relaxation · Camping · Hiking / Outdoors · Road Trip · Business · Event / Conference · Family · Cruise |
| **Trip length** *(optional)* | Any · Weekend (1–3 days) · Short (4–6 days) · Weeklong (7–10 days) · Extended (11+ days) |

### Template → filter mapping

| Template | Planning window | Destination | Trip type | Trip length |
|----------|----------------|-------------|-----------|-------------|
| Trip Planning — 1 Year Out | 1 Year | Any | Any | Any |
| Trip Planning — 6 Months Out | 6 Months | Any | Any | Any |
| Trip Planning — 3 Months Out | 3 Months | Any | Any | Any |
| Trip Planning — 1 Month Out | 1 Month | Any | Any | Any |
| Weekend Getaway | 1 Month | Domestic | General / Mixed, Road Trip | Weekend |
| International Trip | 6 Months | International | Any | Any |
| City & Sightseeing | 3 Months | Any | Sightseeing / City | Any |
| Beach & Relaxation | 3 Months | Any | Beach / Relaxation | Any |
| Family Trip | 3 Months | Any | Family | Any |
| Road Trip | 1 Month | Domestic | Road Trip | Any |
| Business Trip | 1 Month | Any | Business | Any |
| Cruise | 3 Months | Any | Cruise | Any |
| Event / Conference | 1 Month | Any | Event / Conference | Any |
| Camping Trip | 1 Month | Any | Camping | Any |
| Hiking & Outdoors | 1 Month | Any | Hiking / Outdoors | Any |

### Auto-recommendation logic

The app pre-selects the most appropriate template when the modal opens using a **type-first** priority:

1. **Trip type chip selected** — if the user selects a type chip (e.g., "Camping"), the first template whose `filterTripType` matches is auto-selected and highlighted with a *Recommended* badge. Timeline-based templates are deprioritized.
2. **No type selected (timeline fallback)** — based on days until departure and (when available) actual trip duration:

| Condition | Recommended template |
|-----------|----------------------|
| ≥ 300 days until departure | Trip Planning — 1 Year Out |
| 150–299 days until departure | Trip Planning — 6 Months Out |
| 60–149 days until departure | Trip Planning — 3 Months Out |
| Trip is ≤ 3 days long AND departure ≤ 30 days away | Weekend Getaway |
| Otherwise (< 60 days, trip not short) | Trip Planning — 1 Month Out |

Weekend Getaway is only auto-recommended when the trip itself is 1–3 days long — not simply because departure is imminent.

The recommended template updates live as the user clicks filter chips (type, destination, length). Users can always override by clicking any template card.

## Packing Preset Recommendations

When the packing picker opens, presets are sorted by relevance to the current trip's filters. A **Best match** badge highlights the top-scored preset when any filter is active. Scoring:

| Signal | Points |
|--------|--------|
| Trip type matches preset `filterTripType` | +4 |
| Destination matches preset `filterDestination` | +2 |
| Trip length matches preset `filterLength` | +1 |

The planning-project template's trip type is automatically propagated as a pre-filter when the packing picker is opened immediately after project creation.

## Custom Templates

Users can copy any built-in planning or packing template into a custom template for later editing.

**Custom planning project templates** are stored in `project_templates` and support the same filter fields as built-in templates (`filter_trip_type`, `filter_destination`, `filter_length`, `source_id`). Managed via:
- `GET /api/projects/templates` — list all custom planning templates
- `POST /api/projects/templates` — create (copy or from scratch)
- `GET /api/projects/templates/{id}` — fetch one
- `PATCH /api/projects/templates/{id}` — partial update
- `DELETE /api/projects/templates/{id}` — delete

**Custom packing templates** are stored in `packing_templates` and support categories, items, `owner_type` routing, and the same filter fields. Managed via `/api/packing-templates` (list, create, get, put, delete, from-preset, replace).
